# LuaFramework_UGUI_V2 => CSharpLua_UGUI
基于tolua && CSharp.lua的热更新框架

支持C#/Lua双模运行|双模调试
> 开发的时候业务代码都用C#写 底层一点的代码就包装tolua wraper

> 发布的时候业务代码全部转换为lua和预制体一起打包成assetbundle更新

> C#/Lua均可进行代码调试 方便定位问题
